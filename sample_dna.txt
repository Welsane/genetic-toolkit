# Genetic Analyzer Toolkit — Sample DNA Input File
# =================================================
# Lines starting with '#' are treated as comments and are ignored.
# Lines starting with '>' are FASTA-style headers and are also skipped.
# Each remaining line is treated as one DNA sequence.
#
# Format rules:
#  - Only the characters A, T, G, C are allowed (case-insensitive).
#  - For amino acid translation (Option 5), the length must be a multiple of 3.
#  - For mutation detection (Option 4), enter two sequences directly in the menu.
#

About This Project

The Genetic Analyzer Toolkit is a full-stack bioinformatics project. The backend is written in C (C11 standard), structured across main.c, dna_utils.c, and file_handler.c.

This web frontend can operate in two modes: standalone JS simulation (no server required) or connected to a C backend API via fetch() calls — just swap the endpoint constants at the top of app.js.

/* DNA → RNA Transcription */
void dna_to_rna(const char *dna, char *rna) {
  for (size_t i = 0; dna[i]; i++) {
    switch (dna[i]) {
      case 'A': rna[i] = 'U'; break;
      case 'T': rna[i] = 'A'; break;
      case 'G': rna[i] = 'C'; break;
      case 'C': rna[i] = 'G'; break;
    }
  }
  rna[strlen(dna)] = '\0';
}
